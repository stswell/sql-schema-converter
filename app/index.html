AMD SQL Converter

Convert AdvancedMD SQL scripts into PostgreSQL‑ready schemas — safely, locally, and transparently.

Designed for AdvancedMD InterOps ODBC Access Kit scripts targeting PostgreSQL 14 on AWS RDS. Supports both Create Tables and Data Maintenance scripts for PM and EHR modules. Everything runs in your browser — no files are uploaded, no servers are contacted.

Static, client-side conversion — MS SQL in, PostgreSQL out.

✓ What this tool does

  • Parses AMD InterOps script structure and auto-detects the script type
  • Converts MS SQL Server data types, identifiers, and DDL constructs to PostgreSQL 14 equivalents
  • Translates Data Maintenance OPENQUERY / dynamic SQL patterns into direct foreign-table PL/pgSQL
  • Preserves run-history tracking, incremental logic, and table relationships
  • Generates a downloadable conversion report with full transparency on every change

⚠ What this tool does not do

  • Does not execute SQL or connect to any database
  • Does not validate output against a live PostgreSQL instance
  • Does not guarantee production-ready output — always review and test before running
  • Does not modify your source files or persist any data
  • Does not handle MERGE statements, cursors, or complex dynamic branching (these are flagged as warnings)
Best-effort static conversion. Designed to accelerate migration work, not replace human review. Always test output before running in production.
