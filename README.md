# MS SQL → PostgreSQL Schema Converter

This tool converts MS SQL Server 2005 `.sql` schema files into
PostgreSQL 14–compatible syntax.

## How it works
- Runs entirely in your browser
- No files are uploaded to a server
- Upload → Convert → Download

## Usage
1. Open the tool
2. Upload your MS SQL `.sql` file
3. Download the converted PostgreSQL file

## Notes
- Designed for schema conversion (tables, constraints, indexes)
- Review output before executing in production
